Fluid Button Component

This button scales smoothly across all screen sizes using CSS clamp(). Resize your browser window to see how it adapts without any breakpoints.

How It Works

Instead of using breakpoints, the button uses clamp() functions:

  • Horizontal Padding: clamp(0.5rem, 2vw, 1.25rem)
  • Vertical Padding: clamp(0.25rem, 1vw, 0.75rem)
  • Font Size: clamp(0.875rem, 2.5vw, 1.125rem)
  • Icon Size: clamp(1rem, 3vw, 1.75rem)
