Testing Instructions

1. Resize Test: Slowly drag your browser window from mobile (390px) to desktop (1440px+) and watch the smooth scaling.

2. Font Weight Test: Compare the different weights (Regular, Medium, Semibold, Bold) within each size category.

3. Dark Mode Test: Click the moon icon in the top-right to test dark mode typography.

4. Screen Size Monitor: Watch the bottom-left indicator to see exact viewport width.

5. Hierarchy Test: Notice how the visual hierarchy is maintained across all screen sizes.
